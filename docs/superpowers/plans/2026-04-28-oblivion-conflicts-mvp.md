**Module responsibilities:**

- `errors.py` — pure data: error codes, JSON-formatted error emitter to stderr.
- `plugin_source.py` — resolve `--mo2-profile` / `--plugins` / `--plugins-file` to an ordered `[str, ...]` of plugin filenames.
- `xedit_runner.py` — build the xEdit command line, write the load-list tempfile and args sidecar, invoke `subprocess.run`, read result JSON. No business logic about queries — just plumbing.
- `output.py` — wrap query results in the meta envelope, emit JSON to stdout (default) or render the human-format table.
- `cli.py` — argparse, validate args, dispatch to per-command handlers that call into `xedit_runner` and `output`.
- `scripts/*.pas` — one per command + two shared libs. Read args from sidecar JSON, walk records, emit JSON via `lib_json`.

**Tests:**

- Unit tests cover `errors`, `plugin_source`, `xedit_runner` (cmd-line builder only, no subprocess), `output`, `cli` (argparse + dispatch validation only).
- Integration tests run real `obc` against fixture plugins via xEdit headless; skipped if `OBLIVION_CONFLICTS_XEDIT` env var is absent.
- Fixture plugins are generated by `tests/fixtures/_generator/generate_fixtures.pas`, run once via xEdit GUI; resulting `.esp`/`.esm` files committed.

- [ ] **Step 3: Implement `errors.py`**

```python
from __future__ import annotations

import enum
import json
import sys
from dataclasses import dataclass, field
from typing import IO


class ErrorCode(enum.Enum):
    USER_ARG = ("user_arg", 1)
    XEDIT_FAIL = ("xedit_fail", 2)
    PLUGIN_NOT_LOADED = ("plugin_not_loaded", 3)

    def __init__(self, name: str, exit_code: int) -> None:
        self.code_name = name
        self.exit_code = exit_code


@dataclass
class ObcError(Exception):
    code: ErrorCode
    message: str
    details: dict = field(default_factory=dict)

    def __str__(self) -> str:
        return f"{self.code.code_name}: {self.message}"


def emit_error(err: ObcError, *, fmt: str = "json", stream: IO[str] | None = None) -> None:
    out = stream if stream is not None else sys.stderr
    if fmt == "json":
        payload = {
            "error": {
                "code": err.code.code_name,
                "message": err.message,
                "details": err.details,
            }
        }
        out.write(json.dumps(payload) + "\n")
    else:
        out.write(f"error: {err.code.code_name}: {err.message}\n")
        if err.details:
            out.write(f"  details: {err.details}\n")
```

```python
from __future__ import annotations

from pathlib import Path

from oblivion_conflicts.errors import ErrorCode, ObcError


def _read_lines(path: Path) -> list[str]:
    text = path.read_text(encoding="utf-8")
    if text.startswith("﻿"):
        text = text[1:]
    return [line.rstrip("\r") for line in text.split("\n")]


def resolve_from_mo2_profile(profile_dir: Path) -> list[str]:
    """Resolve a load order from an MO2 profile directory.

    Returns plugin filenames in load order, filtered to those marked active
    in plugins.txt (lines prefixed with '*').
    """
    profile_dir = Path(profile_dir)
    loadorder_path = profile_dir / "loadorder.txt"
    plugins_path = profile_dir / "plugins.txt"

    if not loadorder_path.is_file():
        raise ObcError(
            ErrorCode.USER_ARG,
            f"loadorder.txt not found in profile: {profile_dir}",
            {"profile": str(profile_dir), "expected": str(loadorder_path)},
        )
    if not plugins_path.is_file():
        raise ObcError(
            ErrorCode.USER_ARG,
            f"plugins.txt not found in profile: {profile_dir}",
            {"profile": str(profile_dir), "expected": str(plugins_path)},
        )

    enabled: set[str] = set()
    for line in _read_lines(plugins_path):
        s = line.strip()
        if not s or s.startswith("#"):
            continue
        if s.startswith("*"):
            enabled.add(s[1:].strip())

    ordered: list[str] = []
    for line in _read_lines(loadorder_path):
        s = line.strip()
        if not s or s.startswith("#"):
            continue
        if s in enabled:
            ordered.append(s)
    return ordered
```

```python
def resolve_from_explicit(plugins: list[str]) -> list[str]:
    """Resolve from a list passed as CLI args. Strips whitespace, skips blanks."""
    cleaned = [p.strip() for p in plugins if p.strip()]
    if not cleaned:
        raise ObcError(
            ErrorCode.USER_ARG,
            "no plugins given to --plugins",
            {},
        )
    return cleaned


def resolve_from_plugins_file(path: Path) -> list[str]:
    """Resolve from a file with one plugin filename per line. '#' starts a comment."""
    path = Path(path)
    if not path.is_file():
        raise ObcError(
            ErrorCode.USER_ARG,
            f"plugins file not found: {path}",
            {"path": str(path)},
        )
    out: list[str] = []
    for line in _read_lines(path):
        s = line.split("#", 1)[0].strip()
        if s:
            out.append(s)
    if not out:
        raise ObcError(
            ErrorCode.USER_ARG,
            f"plugins file is empty: {path}",
            {"path": str(path)},
        )
    return out
```

from __future__ import annotations

import datetime as dt
import json
import sys
from typing import IO

from oblivion_conflicts import __version__


def build_envelope(
    *,
    command: str,
    args: dict,
    load_order: list[str],
    xedit_version: str,
    duration_ms: int,
    results: list,
) -> dict:
    return {
        "meta": {
            "tool_version": __version__,
            "xedit_version": xedit_version,
            "game": "tes4",
            "command": command,
            "args": args,
            "load_order": load_order,
            "started_at": dt.datetime.now(dt.timezone.utc).isoformat(timespec="seconds"),
            "duration_ms": duration_ms,
        },
        "results": results,
    }


def emit_json(envelope: dict, *, stream: IO[str] | None = None) -> None:
    out = stream if stream is not None else sys.stdout
    out.write(json.dumps(envelope) + "\n")


def emit_human(envelope: dict, *, stream: IO[str] | None = None) -> None:
    """Plain tabular fallback. Best-effort; JSON is the contract."""
    out = stream if stream is not None else sys.stdout
    cmd = envelope["meta"]["command"]
    results = envelope["results"]
    out.write(f"# command: {cmd}\n")
    out.write(f"# results: {len(results)}\n")
    for r in results:
        out.write(json.dumps(r) + "\n")

from __future__ import annotations

import argparse
import sys

from oblivion_conflicts.errors import ErrorCode, ObcError, emit_error


def _common_parent() -> argparse.ArgumentParser:
    p = argparse.ArgumentParser(add_help=False)
    p.add_argument("--xedit-path", required=False,
                   help="Path to TES4Edit.exe; falls back to OBLIVION_CONFLICTS_XEDIT env.")
    p.add_argument("--game-data", required=True,
                   help="Path to Oblivion Data/ directory.")
    src = p.add_argument_group("plugin source (exactly one)")
    src.add_argument("--mo2-profile", default=None,
                     help="MO2 profile dir; reads plugins.txt + loadorder.txt.")
    src.add_argument("--plugins", nargs="+", default=None,
                     help="Explicit plugin filenames in load order.")
    src.add_argument("--plugins-file", default=None,
                     help="File with one plugin per line, in load order.")
    p.add_argument("--format", choices=["json", "human"], default="json")
    p.add_argument("--quiet", action="store_true",
                   help="Suppress xEdit progress on stderr.")
    return p


def parse_args(argv: list[str]) -> argparse.Namespace:
    parser = argparse.ArgumentParser(prog="obc")
    sub = parser.add_subparsers(dest="command", required=True)

    common = _common_parent()

    p_list = sub.add_parser("list", parents=[common],
                            help="Conflicting FormIDs in <plugin> (overview).")
    p_list.add_argument("plugin")

    p_between = sub.add_parser("between", parents=[common],
                               help="FormIDs both <a> and <b> touch (overview).")
    p_between.add_argument("plugin_a")
    p_between.add_argument("plugin_b")

    p_winners = sub.add_parser("winners", parents=[common],
                               help="Winner per FormID for <plugin>.")
    p_winners.add_argument("plugin")

    p_record = sub.add_parser("record", parents=[common],
                              help="Full field-level dump for one or more FormIDs.")
    p_record.add_argument("formids", nargs="+",
                          help="Load-order FormIDs (e.g. 0x1E012345).")

    return parser.parse_args(argv)


def _validate_source(args: argparse.Namespace) -> None:
    sources = [args.mo2_profile, args.plugins, args.plugins_file]
    given = [s for s in sources if s]
    if len(given) == 0:
        raise ObcError(
            ErrorCode.USER_ARG,
            "exactly one of --mo2-profile / --plugins / --plugins-file is required",
        )
    if len(given) > 1:
        raise ObcError(
            ErrorCode.USER_ARG,
            "only one of --mo2-profile / --plugins / --plugins-file may be given",
        )


def main(argv: list[str] | None = None) -> int:
    argv = list(sys.argv[1:]) if argv is None else argv
    try:
        args = parse_args(argv)
        _validate_source(args)
    except ObcError as e:
        emit_error(e, fmt="json")
        return e.code.exit_code
    except SystemExit as e:
        # argparse error path — already wrote to stderr
        return e.code if isinstance(e.code, int) else 2

    # Dispatch placeholder — real handlers wire xEdit in Task 13+
    raise NotImplementedError(f"command not yet wired: {args.command}")

```pascal

unit lib_json;

interface

procedure JsonInit;
procedure JsonOpenObj;
procedure JsonCloseObj;
procedure JsonOpenArr;
procedure JsonCloseArr;
procedure JsonKey(const k: string);
procedure JsonStr(const s: string);
procedure JsonInt(i: int64);
procedure JsonBool(b: boolean);
procedure JsonNull;
procedure JsonRaw(const r: string);
procedure JsonSave(const path: string);

implementation

uses Classes, SysUtils;

var
  buf: TStringList;
  needsComma: array[0..63] of boolean;
  depth: integer;

procedure ResetState;
var i: integer;
begin
  if Assigned(buf) then buf.Free;
  buf := TStringList.Create;
  for i := 0 to High(needsComma) do needsComma[i] := False;
  depth := 0;
end;

procedure JsonInit;
begin
  ResetState;
end;

procedure WriteComma;
begin
  if needsComma[depth] then buf.Add(',');
  needsComma[depth] := True;
end;

procedure JsonOpenObj;
begin
  WriteComma;
  buf.Add('{');
  Inc(depth);
  needsComma[depth] := False;
end;

procedure JsonCloseObj;
begin
  buf.Add('}');
  Dec(depth);
end;

procedure JsonOpenArr;
begin
  WriteComma;
  buf.Add('[');
  Inc(depth);
  needsComma[depth] := False;
end;

procedure JsonCloseArr;
begin
  buf.Add(']');
  Dec(depth);
end;

function EscapeStr(const s: string): string;
var i: integer; c: char;
begin
  Result := '"';
  for i := 1 to Length(s) do begin
    c := s[i];
    case c of
      '"':  Result := Result + '\"';
      '\':  Result := Result + '\\';
      #8:   Result := Result + '\b';
      #9:   Result := Result + '\t';
      #10:  Result := Result + '\n';
      #12:  Result := Result + '\f';
      #13:  Result := Result + '\r';
    else
      if Ord(c) < 32 then
        Result := Result + '\u' + IntToHex(Ord(c), 4)
      else
        Result := Result + c;
    end;
  end;
  Result := Result + '"';
end;

procedure JsonKey(const k: string);
begin
  WriteComma;
  buf.Add(EscapeStr(k) + ':');
  needsComma[depth] := False;  // value follows directly
end;

procedure JsonStr(const s: string);
begin
  WriteComma;
  buf.Add(EscapeStr(s));
end;

procedure JsonInt(i: int64);
begin
  WriteComma;
  buf.Add(IntToStr(i));
end;

procedure JsonBool(b: boolean);
begin
  WriteComma;
  if b then buf.Add('true') else buf.Add('false');
end;

procedure JsonNull;
begin
  WriteComma;
  buf.Add('null');
end;

procedure JsonRaw(const r: string);
begin
  WriteComma;
  buf.Add(r);
end;

procedure JsonSave(const path: string);
var s: string; i: integer;
begin
  s := '';
  for i := 0 to buf.Count - 1 do s := s + buf[i];
  // Use a TStringList just to write a single line without TStringList's
  // automatic line-ending appending.
  with TFileStream.Create(path, fmCreate) do try
    Write(Pointer(s)^, Length(s));
  finally
    Free;
  end;
end;

initialization
  buf := nil;
  depth := 0;

finalization
  if Assigned(buf) then buf.Free;

end.
```

```python
import os
import subprocess
import sys
import tempfile
from typing import Any

from oblivion_conflicts.errors import ErrorCode, ObcError


def run_xedit_query(
    *,
    xedit_path: Path,
    game_data: Path,
    plugins: list[str],
    script_name: str,
    args_payload: dict[str, Any],
    output_path: Path,
    scripts_dir: Path,
    quiet: bool = False,
) -> list:
    """Invoke xEdit headless with the given query script. Returns the parsed JSON output.

    Mechanics:
      - Writes the load list to a temp file.
      - Writes the args sidecar with output_path baked into the payload.
      - Sets OBC_ARGS_FILE env var to the args path.
      - Runs xEdit synchronously.
      - Reads + parses the output JSON.
    """
    output_path = Path(output_path)
    output_path.parent.mkdir(parents=True, exist_ok=True)

    with tempfile.TemporaryDirectory(prefix="obc_") as td:
        td_path = Path(td)
        load_list_file = td_path / "load.txt"
        write_load_list(load_list_file, plugins)

        args_path = td_path / "args.json"
        payload = dict(args_payload)
        payload["output"] = str(output_path).replace("\\", "/")
        write_args_sidecar(args_path, payload)

        script_path = scripts_dir / script_name
        argv = build_xedit_argv(
            xedit_path=xedit_path,
            game_data=game_data,
            load_list_file=load_list_file,
            script_path=script_path,
        )

        env = os.environ.copy()
        env["OBC_ARGS_FILE"] = str(args_path).replace("\\", "/")

        stderr_target = subprocess.DEVNULL if quiet else sys.stderr
        proc = subprocess.run(
            argv,
            env=env,
            stdout=subprocess.DEVNULL,  # xEdit writes JSON to file; stdout noise is irrelevant
            stderr=stderr_target,
        )
        if proc.returncode != 0:
            raise ObcError(
                ErrorCode.XEDIT_FAIL,
                f"xEdit exited with code {proc.returncode}",
                {"argv": argv, "returncode": proc.returncode},
            )
        if not output_path.is_file():
            raise ObcError(
                ErrorCode.XEDIT_FAIL,
                f"xEdit did not produce output file: {output_path}",
                {"argv": argv, "expected_output": str(output_path)},
            )

    return json.loads(output_path.read_text(encoding="utf-8"))
```

function FindFileByName(const name: string): IInterface;
var i: integer; f: IInterface;
begin
  Result := nil;
  for i := 0 to FileCount - 1 do begin
    f := FileByIndex(i);
    if SameText(GetFileName(f), name) then begin Result := f; Exit; end;
  end;
end;

function CollectSubrecordSigs(rec: IInterface; sigs: TStringList): integer;
var i: integer; e: IInterface;
begin
  // Collect direct subrecord signatures (children of the record), counting repeats.
  for i := 0 to ElementCount(rec) - 1 do begin
    e := ElementByIndex(rec, i);
    sigs.Add(Signature(e));
  end;
  Result := sigs.Count;
end;

function CompareSubrecordsRaw(elemA, elemB: IInterface): boolean;
var sA, sB: integer; bytesA, bytesB: TBytes;
begin
  sA := GetDataSize(elemA);
  sB := GetDataSize(elemB);
  if sA <> sB then begin Result := False; Exit; end;
  if sA = 0 then begin Result := True; Exit; end;
  // Compare via xEdit's edit value text — for raw byte access we'd need an
  // API confirmed at impl time. Edit-value equality is a sound proxy because
  // xEdit decodes deterministically.
  Result := SameText(GetEditValue(elemA), GetEditValue(elemB));
end;

function Initialize: integer;
var
  args: TArgs;
  fA, fB: IInterface;
  recsA: TStringList;
  i, j, k: integer;
  rA, rB: IInterface;
  fid, sigA: string;
  diffSigs: TStringList;
  identCount: integer;
  subA, subB: IInterface;
begin
  ArgsLoad(args);
  fA := FindFileByName(args.TargetPluginA);
  fB := FindFileByName(args.TargetPluginB);
  if (not Assigned(fA)) or (not Assigned(fB)) then begin
    AddMessage('plugin not loaded'); Halt(3);
  end;

  // Build map of A's records by load-order FormID.
  recsA := TStringList.Create;
  recsA.Sorted := False;
  try
    for i := 0 to RecordCount(fA) - 1 do begin
      rA := RecordByIndex(fA, i);
      recsA.AddObject(LoadOrderFormIDStr(rA), Pointer(rA));
    end;

    JsonInit;
    JsonOpenArr;
    for i := 0 to RecordCount(fB) - 1 do begin
      rB := RecordByIndex(fB, i);
      fid := LoadOrderFormIDStr(rB);
      j := recsA.IndexOf(fid);
      if j < 0 then Continue;
      rA := IInterface(recsA.Objects[j]);

      diffSigs := TStringList.Create;
      diffSigs.Duplicates := dupIgnore;
      diffSigs.Sorted := True;
      identCount := 0;
      try
        for k := 0 to ElementCount(rA) - 1 do begin
          subA := ElementByIndex(rA, k);
          sigA := Signature(subA);
          subB := ElementBySignature(rB, sigA);
          if not Assigned(subB) then begin
            diffSigs.Add(sigA);
          end else begin
            if CompareSubrecordsRaw(subA, subB) then
              Inc(identCount)
            else
              diffSigs.Add(sigA);
          end;
        end;
        // Subrecords present in B but not in A
        for k := 0 to ElementCount(rB) - 1 do begin
          subB := ElementByIndex(rB, k);
          if not Assigned(ElementBySignature(rA, Signature(subB))) then
            diffSigs.Add(Signature(subB));
        end;

        JsonOpenObj;
          JsonKey('formid');         JsonStr(fid);
          JsonKey('signature');      JsonStr(Signature(rA));
          JsonKey('edid');           JsonStr(EditorID(rA));
          JsonKey('winner_plugin');  JsonStr(GetFileName(GetFile(WinningOverride(rA))));
          JsonKey('differing_subrecords'); JsonOpenArr;
            for k := 0 to diffSigs.Count - 1 do JsonStr(diffSigs[k]);
          JsonCloseArr;
          JsonKey('identical_subrecords_count'); JsonInt(identCount);
        JsonCloseObj;
      finally
        diffSigs.Free;
      end;
    end;
    JsonCloseArr;
    JsonSave(args.OutputPath);
    wbDontSave := True;
  finally
    recsA.Free;
  end;
  Result := 0;
end;

  Result := (GetElementNativeValues(rec, 'Record Header\Record Flags') and $00000020) <> 0;
end;

procedure EmitSubrecord(sig: string; chain: TList; winner: IInterface);
var
  k, idxWinner: integer;
  c, sub, winSub: IInterface;
  isConflict: boolean;
  firstVal: string;
  values: TStringList;  // per-plugin (plugin-name=edit_value) tracking
  hasFirst: boolean;
begin
  values := TStringList.Create;
  try
    hasFirst := False;
    isConflict := False;
    winSub := nil;

    for k := 0 to chain.Count - 1 do begin
      c := IInterface(chain[k]);
      sub := ElementBySignature(c, sig);
      if not Assigned(sub) then Continue;
      values.AddObject(GetEditValue(sub), Pointer(c));
      if not hasFirst then begin firstVal := GetEditValue(sub); hasFirst := True; end
      else if firstVal <> GetEditValue(sub) then isConflict := True;
      if Equals(c, winner) then winSub := sub;
    end;

    JsonOpenObj;
      JsonKey('signature');     JsonStr(sig);
      JsonKey('is_conflict');   JsonBool(isConflict);
      JsonKey('winner_plugin'); JsonStr(GetFileName(GetFile(winner)));
      if Assigned(winSub) then begin
        JsonKey('winner_value'); JsonStr(GetEditValue(winSub));
      end else begin
        JsonKey('winner_value'); JsonNull;
      end;
      JsonKey('values'); JsonOpenArr;
        for k := 0 to values.Count - 1 do begin
          c := IInterface(values.Objects[k]);
          sub := ElementBySignature(c, sig);
          JsonOpenObj;
            JsonKey('plugin');     JsonStr(GetFileName(GetFile(c)));
            JsonKey('edit_value'); JsonStr(values[k]);
            JsonKey('raw_hex');    JsonStr('');  // impl-time: real raw bytes if API allows
            JsonKey('is_winner');  JsonBool(Equals(c, winner));
          JsonCloseObj;
        end;
      JsonCloseArr;
    JsonCloseObj;
  finally
    values.Free;
  end;
end;

procedure EmitRecord(rec: IInterface);
var
  chain: TList;
  winner: IInterface;
  sigs: TStringList;
  i, k: integer;
  c, sub: IInterface;
begin
  chain := GetChainArr(rec);
  sigs := TStringList.Create;
  sigs.Sorted := True;
  sigs.Duplicates := dupIgnore;
  try
    for i := 0 to chain.Count - 1 do begin
      c := IInterface(chain[i]);
      for k := 0 to ElementCount(c) - 1 do begin
        sub := ElementByIndex(c, k);
        sigs.Add(Signature(sub));
      end;
    end;
    winner := WinningOverride(rec);

    JsonOpenObj;
      JsonKey('formid');         JsonStr(LoadOrderFormIDStr(rec));
      JsonKey('signature');      JsonStr(Signature(rec));
      JsonKey('edid');           JsonStr(EditorID(rec));
      JsonKey('master_plugin');  JsonStr(GetFileName(GetFile(IInterface(chain[0]))));
      JsonKey('winner_plugin');  JsonStr(GetFileName(GetFile(winner)));
      JsonKey('chain'); JsonOpenArr;
        for i := 0 to chain.Count - 1 do
          JsonStr(GetFileName(GetFile(IInterface(chain[i]))));
      JsonCloseArr;
      JsonKey('is_deleted');  JsonBool(IsDeleted(rec));
      JsonKey('is_injected'); JsonBool(False);  // see note in query_list
      JsonKey('subrecords'); JsonOpenArr;
        for i := 0 to sigs.Count - 1 do
          EmitSubrecord(sigs[i], chain, winner);
      JsonCloseArr;
    JsonCloseObj;
  finally
    chain.Free;
    sigs.Free;
  end;
end;

function Initialize: integer;
var
  args: TArgs;
  i: integer;
  rec: IInterface;
begin
  ArgsLoad(args);
  JsonInit;
  JsonOpenArr;
  for i := 0 to args.TargetFormIDs.Count - 1 do begin
    rec := FindRecordByFormIDStr(args.TargetFormIDs[i]);
    if not Assigned(rec) then begin
      JsonOpenObj;
        JsonKey('formid'); JsonStr(args.TargetFormIDs[i]);
        JsonKey('error');  JsonStr('not_found');
      JsonCloseObj;
    end else begin
      EmitRecord(rec);
    end;
  end;
  JsonCloseArr;
  JsonSave(args.OutputPath);
  wbDontSave := True;
  Result := 0;
end;

end.

def emit_human(envelope: dict, *, stream: IO[str] | None = None) -> None:
    """Tabular output for terminal use. Drives off the JSON envelope."""
    from io import StringIO
    out = stream if stream is not None else sys.stdout
    cmd = envelope["meta"]["command"]
    results = envelope["results"]

    try:
        from rich.console import Console
        from rich.table import Table
    except ImportError:
        # Plain fallback if rich isn't installed
        out.write(f"# command: {cmd}\n")
        for r in results:
            out.write(json.dumps(r) + "\n")
        return

    buf = StringIO()
    console = Console(file=buf, width=200, force_terminal=False)
    if cmd in ("list", "winners", "between"):
        table = Table(title=f"obc {cmd}  ({len(results)} results)")
        if cmd == "list":
            table.add_column("FormID"); table.add_column("Sig"); table.add_column("EDID")
            table.add_column("Chain"); table.add_column("Winner")
            for r in results:
                table.add_row(r["formid"], r["signature"], r["edid"],
                              " > ".join(r["chain"]), r["winner_plugin"])
        elif cmd == "winners":
            table.add_column("FormID"); table.add_column("Sig"); table.add_column("EDID")
            table.add_column("Winner"); table.add_column("Is winner?")
            for r in results:
                table.add_row(r["formid"], r["signature"], r["edid"],
                              r["winner_plugin"], "yes" if r["is_winner"] else "no")
        else:  # between
            table.add_column("FormID"); table.add_column("Sig"); table.add_column("EDID")
            table.add_column("Differing subrecords"); table.add_column("Identical")
            for r in results:
                table.add_row(r["formid"], r["signature"], r["edid"],
                              ", ".join(r["differing_subrecords"]),
                              str(r["identical_subrecords_count"]))
        console.print(table)
    elif cmd == "record":
        for r in results:
            if "error" in r:
                console.print(f"[red]{r['formid']}: {r['error']}[/red]")
                continue
            console.print(f"[bold]{r['formid']}  {r['signature']}  {r['edid']}[/bold]")
            console.print(f"  chain: {' > '.join(r['chain'])}    winner: {r['winner_plugin']}")
            for sub in r["subrecords"]:
                marker = "[red]CONFLICT[/red]" if sub["is_conflict"] else "[green]agree[/green]"
                console.print(f"  [{sub['signature']}] {marker}  winner={sub['winner_value']!r}")
                if sub["is_conflict"]:
                    for v in sub["values"]:
                        win = " *" if v["is_winner"] else "  "
                        console.print(f"    {win} {v['plugin']:<25} {v['edit_value']!r}")
            console.print()
    else:
        for r in results:
            console.print(json.dumps(r))

    out.write(buf.getvalue())

# Pascal scripts

The `src/oblivion_conflicts/scripts/` directory contains the Pascal scripts that run inside xEdit's script engine. Each command has its own `query_*.pas`; they share `lib_json.pas` (write-only JSON emitter) and `lib_conflicts.pas` (helpers).

## Lifecycle

The Python `xedit_runner` invokes xEdit as:

```
TES4Edit.exe -tes4 -nobuildrefs -IKnowWhatImDoing \
  -D:<game-data-dir> \
  -PSEUDO:<load-list-tempfile> \
  -script:<query>.pas
```

with the env var `OBC_ARGS_FILE` pointing at a temp JSON file describing the request. Each script:

1. `Initialize`: calls `ArgsLoad` to parse the args sidecar.
2. Walks records via xEdit's `RecordCount`/`RecordByIndex` and override-chain API.
3. Emits JSON via `lib_json` to the path in `args.output`.
4. Sets `wbDontSave := True` to skip xEdit's save dialog.
5. Returns 0 from `Initialize`.

## xEdit script API used

| Function | Purpose |
|---|---|
| `MasterOrSelf(rec)` | Originating record (root of override chain). |
| `OverrideCount(rec)` / `OverrideByIndex(rec, i)` | Walk overrides. |
| `WinningOverride(rec)` | Last in load order. |
| `LoadOrderFormID(rec)` | Canonical FormID across plugins. |
| `RecordCount(file)` / `RecordByIndex(file, i)` | Iterate records. |
| `ElementCount(elem)` / `ElementByIndex(elem, i)` | Walk subrecords. |
| `ElementBySignature(elem, sig)` | Look up by 4-char signature. |
| `Signature(elem)` / `EditorID(rec)` | Identity. |
| `GetEditValue(elem)` | Decoded display string (uses xEdit's TES4 schema). |

## Adding a new query

1. Create `query_<name>.pas` modeled on the existing scripts.
2. Add a handler in `src/oblivion_conflicts/handlers.py`.
3. Register the subcommand in `src/oblivion_conflicts/cli.py`.
4. Generate an expected snapshot by running the script against `tests/fixtures/data/`.
5. Add an integration test in `tests/integration/test_e2e.py`.
